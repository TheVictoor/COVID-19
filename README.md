# Covid-19

A report UI, created like a progressive web app (PWA), to share covid-19 updated information.

living at [Covid 19](https://dashboard-covid.herokuapp.com/)

## :wrench: Project setup
```dash
  npm install
  npm run serve
```

## :hammer: Project stack  
- Vue.js
- Bootstrap 4
